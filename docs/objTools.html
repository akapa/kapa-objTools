<static> construct(obj, func) → {Object}

Constructs an object from a prototype object and a "constructor" function. Used to create objects that are "named" in the browser's inspector.
Parameters:
Name Type Description
obj Object The object to be used as a prototype.
func function The "constructor" function. It will not be called, it is only used to make the object "named".
